Project Overview

For this complex enterprise project, I architected and developed a multi-layered analytics platform that combines traditional business intelligence with cutting-edge AI capabilities. The system processes data from a Learning Management System containing 40+ interconnected tables and makes it accessible through both interactive dashboards and natural language conversations.

The chatbot serves as an intelligent assistant embedded directly in Power BI, allowing users to ask questions about courses, get personalized recommendations, and access complex data insights without writing SQL or navigating multiple report pages.

Technical Architecture

1 Data Analysis & Modeling

  • Analyzed 40+ interrelated tables from Learning Management System database to identify key entities and relationships
  • Engineered Galaxy schema model optimized for both analytical queries and chatbot performance
  • Denormalized critical data into single table for AI model processing while maintaining normalized structure for reporting

2 Dual-Model AI Architecture

SQL-Based Model

  • • Natural language to SQL query translation
  • • Handles structured data queries
  • • Returns precise, fact-based answers
  • • Optimized for numeric/statistical queries

Vector Embedding Model

  • • Semantic search capabilities
  • • Understands context and intent
  • • Finds conceptually similar content
  • • Handles ambiguous/exploratory queries

Innovation: Both models process queries in parallel, results are intelligently merged and summarized by Azure OpenAI to provide the most comprehensive answer.

3 ChatGPT-like Interface in Power Apps

  • Designed custom chat interface that replicates ChatGPT UX with message bubbles, streaming responses, and typing indicators
  • Implemented session-based chat history allowing users to revisit and continue previous conversations
  • Built Power Automate flows for API orchestration, response handling, and data persistence
  • Overcame Power Apps limitations through creative workarounds for features not natively supported

4 Power BI Integration

Seamlessly embedded the chatbot interface directly inside the Power BI dashboard, creating a truly interactive analytics and AI solution where users can:

  • Ask questions about courses while viewing dashboards
  • Get personalized course recommendations
  • Query historical enrollment data conversationally
  • Discover content based on interests and history

Chatbot Capabilities

🎓 Intelligent Course Discovery

Suggests courses based on user interests, previous education history, and career goals using semantic understanding of course content.

💬 Natural Language Querying

Handles both structured SQL queries ("How many students enrolled last month?") and conceptual questions ("What's the best course for data science beginners?").

🔍 Contextual Search

Vector embeddings enable finding conceptually related courses even when exact keywords don't match, improving discovery.

📊 Session Management

Maintains conversation context across sessions, allowing users to return to previous discussions and build on prior recommendations.
