Key Capabilities

Intelligent Course Discovery

Suggests courses based on user interests, previous education history, and career goals using semantic understanding of course content.

Natural Language Querying

Handles both structured SQL queries ("How many students enrolled last month?") and conceptual questions ("What's the best course for data science beginners?").

Contextual Search

Vector embeddings enable finding conceptually related courses even when exact keywords don't match, improving discovery.

Session Management

Maintains conversation context across sessions, allowing users to return to previous discussions and build on prior recommendations.

Data Architecture

Analyzed 40+ interrelated tables from the Learning Management System database to identify key entities and relationships. Engineered a galaxy schema model optimized for both analytical queries and chatbot performance.

  • Dual-purpose design: normalized for reporting, denormalized for AI model training
  • Vector embedding tables for semantic search and recommendation engine

Dual-Model AI Architecture

SQL-Based Model

  • • Natural language to SQL query translation
  • • Handles structured data queries
  • • Returns precise, fact-based answers

Vector Embedding Model

  • • Semantic search capabilities
  • • Understands context and intent
  • • Handles ambiguous/exploratory queries

Innovation: Both models process queries in parallel, results are intelligently merged and summarized by Azure OpenAI to provide the most comprehensive answer.

Power Apps Integration

  • • Designed custom chat interface replicating ChatGPT UX with message bubbles and streaming responses
  • • Implemented session-based chat history allowing users to revisit previous conversations
  • • Built Power Automate flows for API orchestration and response handling
  • • Embedded chatbot directly inside Power BI dashboard for seamless analytics + AI experience
