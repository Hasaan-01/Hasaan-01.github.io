Project Overview

Designed and developed a custom Leave Request Application using Microsoft Power Apps to streamline the organization's leave management process. The solution completely digitizes request submission, approval workflows, and balance tracking, providing employees and HR with a transparent, efficient, and user-friendly experience.

Prior to this system, leave requests were handled through email chains and manual Excel tracking, leading to lost requests, miscalculated balances, and delayed approvals. This automated solution eliminated these pain points while providing audit trails and compliance documentation.

Key Features

Streamlined Multi-Level Approval Workflow

Automated routing of requests through a defined approval hierarchy: Employee → Line Manager → HR Manager. Each approver receives email notifications and can approve/reject directly from their mobile device or desktop.

Technical Implementation: Power Automate flows trigger on form submission, route to approvers based on organizational hierarchy stored in SharePoint, and update status in real-time.

Real-Time Leave Quota Tracking

Integration with backend database dynamically calculates and displays available leave balances. System automatically deducts approved leave and prevents requests that exceed available quota.

  • Separate tracking for Sick, Casual, and Bereavement leave types
  • Visual indicators (progress bars) showing remaining days
  • Historical usage patterns and accrual tracking

Smart Date Logic & Conflict Detection

Intelligent business logic automatically excludes weekends and public holidays from leave calculations, ensuring accurate day counts.

Conflict Prevention

Validates against overlapping requests and team capacity limits

Holiday Calendar Integration

Syncs with organizational holiday calendar for accurate calculations

Automated Notifications & Transparency

Email alerts triggered at each stage of the approval process, keeping all stakeholders informed in real-time.

  • Employee confirmation on submission
  • Manager notification for pending approval
  • HR alert for final approval
  • Employee notification on approval/rejection

Employee Self-Service Dashboard

Comprehensive dashboard where employees can view leave balances, request history, upcoming company holidays, and pending approval status.

Mobile-First Design: Fully responsive interface ensures employees can submit and track requests from any device, anywhere.
