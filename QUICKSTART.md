# ⚡ Quick Start

## View Locally (Right Now!)

Just double-click `index.html` to open it in your browser. That's it!

## Deploy to GitHub (5 Minutes)

### 1. Update Your Contact Info
Edit `index.html` (search for "your.email@example.com" and update):
- Line ~454: Your email
- Line ~462: Your LinkedIn URL

### 2. Push to GitHub
```bash
cd C:\Users\Hasaannn\.openclaw\workspace\hasaan-portfolio

git init
git add .
git commit -m "Initial commit"

# Replace YOUR_USERNAME with your GitHub username
git remote add origin https://github.com/YOUR_USERNAME/hasaan-portfolio.git
git push -u origin main
```

### 3. Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Source: **GitHub Actions**
3. Wait 2-3 minutes
4. Visit: `https://YOUR_USERNAME.github.io/hasaan-portfolio/`

Done! 🎉
